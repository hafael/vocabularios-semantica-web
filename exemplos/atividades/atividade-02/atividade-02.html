Atividade 02: Criando Triplas de Dados

Do VS Code ao Servidor Fuseki

Objetivo

Praticar a criação de um arquivo de dados em formato Turtle (.ttl) utilizando o vocabulário SKOS e Dublin Core, e publicá-lo em um servidor triplestore (Fuseki).

Passo 1: Criação do Arquivo RDF (VS Code)

Tarefa 1

Novo Arquivo Turtle

  1. Abra o Visual Studio Code.
  2. Vá em File > New Text File.
  3. Salve o arquivo imediatamente (Ctrl+S ou Cmd+S) com o nome: treino.ttl.
IMPORTANTE: A extensão .ttl é fundamental para que o sistema reconheça o formato Turtle.
Tarefa 2

Definição do Código

Copie e cole o código abaixo no seu arquivo treino.ttl:

@prefix skos: <http://www.w3.org/2004/02/skos/core#> .
@prefix dc: <http://purl.org/dc/elements/1.1/> .
@prefix ex: <http://minhaaula.org/id/> .

# O ALICERCE: Criando o conceito de "Naturalismo"
ex:Naturalismo a skos:Concept ;
    skos:prefLabel "Naturalismo"@pt ;
    skos:altLabel "Escola de Zola"@pt .
 
# A PONTE: Catalogando um livro e ligando ao conceito
ex:livro001 a dc:Text ;
    dc:title "O Cortiço" ;
    dc:creator "Aluísio Azevedo" ;
    dc:subject ex:Naturalismo .

Salve o arquivo novamente após colar.

Passo 2: Publicação no Servidor (Fuseki)

Tarefa 3

Acessar o Fuseki

Certifique-se de que os servidores estão ligados (start.bat) e acesse:

http://localhost:3030

Login: admin / Senha: admin

Tarefa 4

Criar Dataset e Carregar Dados

  1. No menu superior, clique em "Manage Datasets".
  2. Clique em "Add one".
  3. No campo Dataset name, digite: TesteAluno.
  4. Escolha o tipo Persistent e clique em Create dataset.
  5. No menu lateral, vá em "Upload Data".
  6. Clique em "Select files" e selecione o arquivo treino.ttl que você salvou.
  7. Clique no botão verde "Upload now".
CHECKPOINT: Se aparecer uma mensagem de Success com fundo verde, seus dados foram publicados com sucesso!

Entrega

Mostre ao professor o resultado da aba "Info" do seu dataset TesteAluno no Fuseki, comprovando o carregamento dos dados.
