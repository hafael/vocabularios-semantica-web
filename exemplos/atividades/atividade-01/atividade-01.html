Atividade 01: Configuração do Laboratório

Preparação do ambiente de trabalho para a Web Semântica

Objetivo

Instalar as ferramentas necessárias e iniciar os servidores (Fuseki e WebProtégé) que serão utilizados durante todo o curso.

Passo a Passo

Tarefa 1

Obter os arquivos do projeto

Baixe o repositório do projeto no GitHub como um arquivo ZIP e extraia em uma pasta de fácil acesso (ex: C:\TrabalhoSemantico no Windows ou ~/Documents/TrabalhoSemantico no macOS).

Tarefa 2

Instalar o Motor (Docker Desktop)

Instale o Docker Desktop. Se você usa Windows, certifique-se de que o WSL 2 está atualizado conforme as instruções do manual.

DICA: O Docker deve estar aberto e exibindo o ícone da baleia verde na barra de tarefas antes de prosseguir.
Tarefa 3

Ferramentas de Edição

Instale o Visual Studio Code e o MarcEdit 7.

Tarefa 4

Subir os Servidores

No Windows, execute o arquivo start.bat. No macOS, use o terminal para rodar o ./start.sh.

CHECKPOINT: Aguarde a mensagem Ambiente pronto! no terminal.
Tarefa 5

Configuração de Administrador do WebProtégé

Abra o terminal e execute o comando para criar sua conta administrativa:

docker exec -it webprotege java -jar /webprotege-cli.jar create-admin-account

Siga as instruções na tela para definir seu e-mail e senha.

Tarefa 6

Validação Final

Acesse as interfaces e verifique se o login funciona:

  • Fuseki: http://localhost:3030 (admin/admin)
  • WebProtégé: http://localhost:5000 (com a conta que você criou)

Entrega

Ao finalizar, mostre ao professor as duas interfaces abertas no seu navegador e os indicadores de status "Online" no arquivo instrucoes.html.
